## 使用

### 使用命令行
```bash
$ git clone https://github.com/ant-design/ant-design-pro.git --depth=1
$ cd ant-design-pro
$ npm install
$ npm start         # 访问 http://localhost:8000
```

### 部署
```bash
git地址
	git@47.107.158.132:platform/spider_client.git
  
环境要求
	node >=10.15.3
	cnpm
		npm install -g cnpm --registry=https://registry.npm.taobao.org

构建脚本
	cnpm i
	npm run build
```

构建完成后会在根目录生成dist目录，把dist目录部署到技术部的测试服务器
推送目录： WEB-ROOT/spider/dataeyes/public

参考地址
	https://www.jianshu.com/p/90c04ceff2da
